1.数据结构和算法总览

数据结构分为一维, 二维, 特殊.
一维:
- 基础: 数组array(string), 链表 linked list
- 高级: 栈stack, 队列queue, 双端队列deque, 集合set, 映射map(hash or map), etc
二维:
- 基础: 树tree, 图graph
- 高级: 二叉搜索树binary search tree(red-black tree, AVL), 堆heap, 并查集disjoint set, 字典树Trie, etc
特殊:
- 位运算Bitwise, 布隆过滤器BloomFilter
- LRU Cache

算法:

  • if-else, switch -> branch
  • for, while loop -> lteration
  • 递归Recursion(Divide & Conquer, Backtrace)
  • 搜索Search: 深度优先搜索Depth first search, 广度优先搜索Breadth first search, A*, etc
  • 动态规划Dynamic Programming
  • 二分查找 Binary Search
  • 贪心 Greedy
  • 数学 Math, 集合Geometry

1. 刻意练习, 过遍数(五毒神掌)
2. 练习缺陷, 弱点地方
3. 不舒服, 不爽, 枯燥表示在进步
4. 主动反馈: 自己看别人代码, 比如github, LeetCode
5. 被动反馈: 高手指点你, 比如code review

切题四件套:

  1. 看清题目, 不理解直接沟通清楚这个题目问什么
  2. 想所有可能的想法去解题, 时间复杂度+空间复杂度
  3. 用最优思路写代码
  4. 测试样例多举几个

五毒神掌
第一遍:

  1. 5分钟: 读题+思考
  2. 会, 尝试书写
  3. 不会, 直接看解法: 多种解法, 比较优劣
  4. 背诵, 默写好的解法

第二遍:

  1. 自己写, 尝试多种写法

第三遍: 24小时之后

第四遍: 过了一周之后
